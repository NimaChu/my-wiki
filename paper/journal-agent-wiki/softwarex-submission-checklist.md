# SoftwareX Submission Checklist

Checked against the official SoftwareX Guide for Authors on 2026-07-15.

## Manuscript

- [x] Article type is positioned as Original Software Publication.
- [x] Uses the official SoftwareX Original Software Publication LaTeX structure.
- [x] Contains the five mandatory sections:
  Motivation and significance, Software description, Illustrative examples,
  Impact, and Conclusions.
- [x] Abstract is below 250 words.
- [x] Uses no more than seven keywords.
- [x] Includes mandatory code metadata table.
- [x] Includes optional executable software metadata table.
- [x] Includes data availability, CRediT, funding, competing-interest, and AI-use
  declarations.
- [x] Highlights are supplied as a separate editable file with five bullets.
- [x] No AI-generated figure or graphical abstract is included.
- [x] PDF contains approximately 1914 words in total, including metadata tables
  and references, so the running text is below the current 3000-word limit.
- [x] Corresponding author's complete postal address is included.
- [x] Corresponding author's phone number has been provided for Editorial
  Manager and is intentionally not stored in the public manuscript repository.

## Public Software Repository

- [x] Public GitHub repository exists.
- [x] README.md exists and describes installation and use.
- [x] Permanent release tag v0.2.0 exists.
- [x] GitHub Release and Zenodo DOI exist.
- [x] MIT LICENSE.txt has been added.
- [x] Canonical public CLI source is under the top-level src/ directory; legacy
  scripts/ paths remain as compatibility wrappers.
- [x] Created a permanent release after license and repository-layout changes.
- [x] Updated the manuscript metadata tables to v0.2.0 and its Zenodo DOI.
- [ ] Consider publishing a sanitized case-study vault or reproducible fixture.

## Submission Files

- [x] SoftwareX manuscript source: softwarex.tex.
- [x] Compiled SoftwareX PDF.
- [x] Highlights file: highlights.txt.
- [x] SoftwareX cover letter: cover-letter.md.
- [x] Complete LaTeX source ZIP after compilation.
- [ ] Elsevier declarations-tool DOC/DOCX file for competing interests.
- [ ] Optional graphical abstract intentionally omitted.

## Financial and Policy Notes

- SoftwareX is an open-access journal with an article publishing charge if the
  paper is accepted. Recheck the current amount immediately before submission.
- The journal requires disclosure of generative AI used in manuscript
  preparation. The manuscript contains that disclosure.
- The local case-study corpus is not currently public. The data statement is
  explicit about this limitation.
