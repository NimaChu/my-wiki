# Knowledge Index

This is the main entry point for the vault.

## Reading Path

1. Search this page and the most relevant wiki topic first.
2. Inspect linked raw notes only when claims need evidence.
3. Use `npm run wiki:search -- "query terms"` for cross-layer search.
4. Use `npm run dashboard` for graph browsing and health checks.

## Starter Pages

Add durable wiki pages here as the vault grows.

## Maintenance

- Review `[[log]]` after each ingest or substantial wiki edit.
- Run `npm run wiki:status`, `npm run wiki:lint`, and `npm run wiki:garden` regularly.
- Keep orphaned pages and broken links visible until intentionally resolved.
