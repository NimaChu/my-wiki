# Templates

Use these templates when adding sources or compiling wiki pages.

- `raw-source.md`: webpage, article, PDF, and long-form source capture.
- `wiki-topic.md`: broad subject pages.
- `wiki-concept.md`: small atomic ideas.
- `wiki-person.md`: people and their ideas.
- `wiki-company.md`: company/entity profile.
- `wiki-product.md`: product/service profile.
- `wiki-method.md`: reusable method or workflow.
- `wiki-comparison.md`: structured comparison note.
- `web-clipper-karpathy.md`: template content for Obsidian Web Clipper.
